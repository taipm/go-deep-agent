# Archive

This directory contains historical documentation from previous development phases.

## Structure

- **assessments/** - Old capability assessments and architecture analyses (v0.5.6 - v0.6.0)
- **planning/** - Planning documents, implementation plans, and roadmaps  
- **summaries/** - Completion summaries from various development phases
- **evaluations/** - Usability evaluations, design decisions, and tool assessments
